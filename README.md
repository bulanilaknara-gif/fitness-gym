# Fit Life Gym Website

## Project Description
Fit Life Gym is a modern fitness website designed to showcase gym services, trainers, pricing plans, and contact information.  
This website is fully responsive and includes smooth animations and dark/light mode functionality.

---

## Tech Stack
- React JS
- Vite
- CSS3
- AOS Animation Library
- JavaScript (ES6)

---

## Setup Instructions

1. Clone the repository
git clone https://github.com/bulanilaknara-gif/fitness-gym.git

2. Navigate to project folder
cd fitness-gym

3. Install dependencies
npm install

4. Run the project
npm run dev
